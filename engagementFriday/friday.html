What is the difference between a block and an inline tag? Give examples of each one.
Block tags or block elements creates blocks in a page and essentially dividing the page into different segments. Some examples are <div>,<p>,<footer>,etc.
What is normal flow in HTML?
It is the default way that html organizes elements in the webpage. An example would be <header>, <nav>, <span> , etc.
What does float do? Why do you need to clear the float?
Float is a CSS property used to change position of elements into side-by-side, instead of the default stacking behavior.
What do absolute, fixed, and relative do?
Absolute, fixed, and relative are the assigned text/element positioning.
