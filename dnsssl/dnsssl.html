DNS and SSL

1. What is DNS?
It is the phonebook of the internet. it is used to translate readable domain names into ip addresses
2. How do you get a domain name?
You can register for a domain using providers. GoDaddy, Namecheap, Google domains, and domain.com are popular ones
3. What is SSL?
Secure Socket Layer. It is a security protocol that makes a encrypted connection between communicating machines so that its not intercepted by third party attacks.
4. How do you get a security certificate?
Choose a security authority, then purchase that appropriate security needed. As you submit a security request, the certificate authority will evaluate your domain and issue the certificate. then all you have to do is install the certificate on the server.
5. How does SSL work?
first the browser will request the security certificate, then the server sends the certification. The browser will authenticate the certificate if its good to go. if its valid, the browser will generate a shared encrypted connection.
6. How do you stay safe online.
Strong, unique passwords will help lock your data to yourself. antivirus could detect third party attacks and avoid malware as well. Secure Wi Fi networks should be picked over public networks. Public networks are easier to hack.
