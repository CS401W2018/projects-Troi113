1. Internet!

A network of computers that is connected globally. The infrastructure of the internet links computers and allows communicate and access information.

2. Web

It is a service that operates on top of the internet. It is a collection of data that could be accessed via the internet.

3. Internet vs Web

The internet and the web is different in the sense that Internet is the interconnected computer network, while the Web is the interlinked documents and multimedia that is accessed using the web.

4. ASCII and Unicode

ASCII is the set of encoding numbers that are used to represent specific characters. Unicode is a newer standard that is used to accomodate more characters.

5. What is UTF-8

It is the most popular type of unicode for encoding. It uses around one to four bytes each character depending on the complexity of it.

6. URL

The URL or Uniform Resource Locator of a website is its address in the big scope of the Web. We use it to locate data such as documents, videos, websites, etc.

7. Domain Name

A Domain Name represents an IP address. All domain names are registered and managed on to the DNS or Domain Name System.

8. HTTP

Hypertext Transfer Protocol is the protocol that governs the communication between web browsers and servers.

9. HTTPS

Hypertext Transfer Protocol Secure is a more stable and secure version of HTTP. It encrypts data for information protection between two browsers.

10. IP Address

The Internet Protocol Address is a number label that is specifically assigned to a device. We use this to both identify and locate one out of every device in a network.

11. DNS

The Domain Name System is a database that contains databases that translates different domain names into complex IP Addresses

12. W3C Recommendations

To avoid compatibility and interoperability problems between web technologies, the W3C set guidelines for rules, languages and practices around web technologies.

13. XML, XHTML, HTML, HTML5

XML-Extensible Markup Language is a general-purpose language used to define and store data. Used mostly for data exchange and configuration.

XHTML-Extensible Hypertext Markup Language, is a more secure version of HTML, but is based on XML rules.

HTML-Hypertext Markup Language is a standard markup language that is used in coding web pages.

HTML5- is the latest version of HTML used in modern website creation
