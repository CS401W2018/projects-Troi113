CSS Basics

  1. Explain the difference between inline, embedded, and external CSS. - In inline, the design will be applied through typing directly inside the HTML element. In embedded, the design elements are all defined in a style tag place within the header. Lastly, external CSS is defined in a separate .css file. It is usually linked by the html file using what's called a link tag.
  2. Explain why you should only use external CSS. - because using the other html styling techniques are too tedious. CSS provides a less complicated way to style HTML because it is organized.
  3. What is a selector used for? - CSS uses selectors to specify individual HTML elements so that it can differentiate separate styles used.
