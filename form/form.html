1. The form tag has an "action" attribute. What does it do?
The action attribute defines the link where all of the form data is passed and submitted.

2. The form tag has a "method" attribute. What does it do?
The method tag determines the https method that the webpage will use to communicate data to the server. The most common ones are get and post.

3. What is the difference between a "POST" and a "GET"?
"GET" puts the input into the query string. Every data will be visible at the address bar. This makes it unsuitable for sensitive and private data.
"POST" is a more secure choice. It only sends the data but hides it from the browser's address bar. It is more commonly used when dealing with registration, usernames, passwords, files,etc.
