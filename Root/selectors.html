Engagement - CSS Selectors

1. List out the different ways to select an item.

  • Simple Selectors
  • Universal Selector
  • Attribute Selectors
  • Pseudo-Class Selector
  • Combinator Selectors
  • Pseudo-Element Selector

2. What is the difference between an ID and a Class?

ID is used to identify a single element in a document and should only be used once per page.
Class is used to apply styles to multiple parts of the webpage.

3. What is the difference between a SPAN tag and a DIV tag

SPAN should be used to group inline content. The difference is that DIV is a block-level, which creates a new block of code or division within a page.

4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

As explained earlier, DIV is a block-level code which creates a new block of code or division within a page.
A header is typicaly used as for a website's title and navigation.
The Main represents the body of the page and all of its contents.
Lastly, the Footer serves as a copyright and contact section of a website.

5. What are different ways to select a color?

  • Hexadecimal Color Code
  • RGB (Red, Green, Blue) Color Model
  • HSL (Hue, Saturation, Lightness) Color Model
  • Named Colors
